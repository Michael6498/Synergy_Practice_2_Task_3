# Synergy_Practice_2_Task_3
# Информационная система «Бот-помощник ООО «Черник-В»»
### Кейс-задача № 3 по производственной практике

## Описание проекта
Данный проект представляет собой информационную систему на базе Telegram-бота, разработанную для компании «Черник-В» (сфера аренды коммерческой недвижимости). Система предназначена для первичной консультации клиентов, автоматического подбора свободных помещений для аренды и сбора контактных данных с целью снижения нагрузки на менеджеров коммерческого отдела.
В основе логики распознавания запросов лежит модуль NLP (стемминг), позволяющий боту понимать естественную речь пользователя вне зависимости от падежей и форм слов.

## Стек технологий и архитектура
* **Язык программирования:** Python (основная логика приложения)
* **Интерфейс взаимодействия (UI):** Telegram Bot API / библиотека `pyTelegramBotAPI`
* **Обработка естественного языка (NLP):** Библиотека `NLTK`, алгоритм `SnowballStemmer` (морфологический анализ и стемминг текста)
* **Асинхронность и сеть:** Модули `asyncio` и `aiohttp` (конкурентное тестирование и подбор рабочих прокси-серверов)
* **Database / Storage:** JSON (файловое хранилище данных о свободных объектах недвижимости и конфигурации компании).

## Функциональные возможности
* **Анализ текстовых запросов:** распознавание темы сообщения пользователя с помощью регулярных выражений и стемминга для распределения по 5 сценариям (приветствие, контакты, график работы, поиск недвижимости, отправка заявки).
* **Интерактивный подбор недвижимости:** фильтрация объектов из базы данных в реальном времени с выводом актуальных параметров (площадь, этаж, тип, цена).
* **Валидация пользовательского ввода:** пошаговый сценарий для сбора номеров телефонов с автоматической проверкой формата (+7/8 и количество цифр).
* **Отказоустойчивость сетей:** встроенный алгоритм конкурентного тестирования пула прокси-серверов для стабильного обхода сетевых ограничений.
* **Логирование:** фиксация входящих заявок в консоли сервера.
